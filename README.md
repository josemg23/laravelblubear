# prueba-bluebear
Se realiza la prueba de BlueBear

Proyecto realizado en laravel 10 

Complementos utilizados
- laravel 10.x.x
- laravel brezze
- vuejs 3.x.x
- Tailwind 


Comandos para levantamiento del Proyecto.

  - npm install 
  - npm run dev o npm run build  
  - composer install
  - php artisan migrate
  - php artisan key:generate
